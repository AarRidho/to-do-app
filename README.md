## How to

Terdapat 2 cara untuk menjalankan aplikasinya, dengan Vite atau Static Files

## Vite
Untuk menggunakan vite, anda harus menginstall node.js untuk package manager-nya

1. Install Node.js
2. `npm i -g yarn`
3. `yarn install`
4. `yarn dev`

## Static Files

1. Buka file dist
2. Buka file `index.html` di browser
